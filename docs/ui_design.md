# iEditWeb UIデザインガイドライン

## デザインコンセプト: "Card Material" (Light Theme)
物理的な「カード」が重なり合うような奥行きと、整理された清潔感を重視したデザイン。
グラデーションは使用せず、単色（ソリッドカラー）とドロップシャドウで階層を表現する。

### 1. カラーパレット
- **ベース背景色**: 薄いグレー (`#F3F4F6` / `slate-100`) - アプリ全体の背景
- **サーフェス色**: 純白 (`#FFFFFF`) - エディタ、フローチャート、サイドバーのカード背景
- **テキスト色**: 濃いグレー (`#1F2937` / `gray-800`) - 本文
- **アクセント色**: ティール (`#0D9488` / `teal-600`) - ボタン、アクティブ状態、リンク
- **ボーダー色**: 極薄いグレー (`#E5E7EB` / `gray-200`)

### 2. レイアウト構造
画面全体を「背景」とし、各機能エリアを独立した「カード」として配置する。

- **ヘッダー**: 上部に固定。背景色と一体化させるか、薄い白のカードとして浮かる。
- **サイドバー（左）**: 独立した白いカード。角丸（Rounded-lg）。ソフトな影（Shadow-sm）。
- **メインエリア（右）**:
    - **上半分（フローチャート）**: 独立した白いカード。単色背景。
    - **下半分（エディタ）**: 独立した白いカード。単色背景。

### 3. UIコンポーネントスタイル
- **ボタン**: フラットデザイン。塗りつぶし（アクセント色）またはアウトライン。角丸は少し大きめ（Rounded-md）。
- **影（Shadow）**: 拡散する柔らかい影を使用し、浮遊感を出す。
- **タイポグラフィ**: システムフォント（Inter, YuGothic等）を使用し、可読性を最優先。

### 4. インタラクション
- カードにホバーした際、わずかに影を濃くして「触れる」感覚を演出する。
- ボタンクリック時にリップルエフェクト（波紋）または明確な押し込みアニメーションを入れる。
